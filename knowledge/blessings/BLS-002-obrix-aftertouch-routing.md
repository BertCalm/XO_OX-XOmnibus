# BLS-002: OBRIX Aftertouch as 8-Target Performance Grammar

**Ghost**: Vangelis
**Engine**: OBRIX (2026-03-19)

Vangelis blessed OBRIX's decision to wire Aftertouch as a modulation source routable to any of 8 targets (Pitch, FilterCutoff, FilterReso, Volume, WavetablePos, PulseWidth, EffectMix, Pan). "This is not a gimmick — this is a conversation between the hand and the machine." This routing architecture should be preserved and highlighted in presets — at least 3 factory presets should have aftertouch already wired to a meaningful target to demonstrate the instrument's expressive grammar.
