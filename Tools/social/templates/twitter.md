# Twitter Templates

Variables available in all templates:
  {recipe_name}        — coupling recipe name
  {engine_a}           — Engine A short name (e.g. ONSET)
  {engine_b}           — Engine B short name (e.g. OVERBITE)
  {engine_a_full}      — Engine A full description
  {engine_b_full}      — Engine B full description
  {coupling_type}      — primary coupling type (e.g. Amp->Filter)
  {amount}             — coupling amount (e.g. 0.50)
  {direction}          — A→B or bidirectional
  {why_it_works_short} — truncated explanation (≤160 chars)
  {spotlight_preset}   — this week's featured preset name
  {new_preset_count}   — number of new presets this week
  {scripture_verse}    — this week's Guru Bin quote
  {changelog_summary}  — bullet list of commits
  {week_id}            — ISO week (e.g. 2026-W12)
  {week_num}           — sequential week counter
  {reddit_link}        — Directional Reef link to Reddit
  {discord_link}       — Discord invite link
  {breadcrumb}         — first breadcrumb note from this week
