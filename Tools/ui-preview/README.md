# XOceanus UI Preview Server

Hot-reloading HTML/CSS prototypes for XOceanus UI components.
Edit the HTML, save, browser updates instantly.

## Usage
```bash
cd Tools/ui-preview
npx live-server --port=3333
```
Then open http://localhost:3333 in your browser.

## Workflow
1. Edit any .html file in this directory
2. Browser auto-refreshes on save
3. When the design looks right, translate to JUCE paint() calls
