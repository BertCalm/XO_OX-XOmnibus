How the Metaphor Carries Through Every Phase

Phase 1A — The Core Loop
Drop grain → Oyster classifies → nacre layers (velocity) → pearl emerges → export XPN

Phase 1B — Rebirth Mode
The Oyster's biology IS the engine. When a grain passes through OVERWASH, the nacre is shaped by the engine's character. Different oysters (engines) produce different pearls from the same grain.

Phase 1C — Design Mode
Manual pearl shaping — you open the shell and adjust the layers by hand. Zone editing = choosing where each nacre layer sits. Expression mapping = how the pearl responds to touch.

MPE Connection
The Seaboard doesn't just play the pearl — it reaches inside the shell. Slide changes the nacre angle (filter). Pressure compresses the layers (dynamics). The pearl is alive because the oyster built it to respond.
