2. MPE Expression Mapping (Phase 1A — Critical)

Every pearl ships with MPE expression pre-wired. The Seaboard plays it immediately.

Melodic Defaults
Slide (CC74) → Filter cutoff sweep
Pressure → Filter resonance + amp boost
Pitch bend → ±48 semitones (Seaboard range)
Velocity → Amplitude + brightness (spectral tilt)
Drum Defaults
Slide (CC74) → Pitch offset (±2 semi)
Pressure → Choke speed
Pitch bend → Disabled (one-shots)
Velocity → Amplitude + saturation drive

Expression defaults are category-aware. Manual override available in Phase 1C Design Mode.

3. XPN Output Spec (MPC Hardware Compatibility)

Keygroup programs:
• KeyTrack=True (always) • RootNote from autocorrelation (not hardcoded 60)
• VelStart=0 on empty layers (MPC requirement) • Up to 4 velocity layers
• MPE: slide→CC74 filter, pressure→aftertouch filter depth, bend range=48

Drum programs:
• GM note mapping (kick=C2, snare=D2, hat=F#2, clap=D#2)
• Choke groups: closed hat mutes open hat (auto-detected)
• One-shot=True for all drum hits • Mute groups for hats

WAV specs:
• 44.1kHz or 48kHz (match source, never upsample) • 24-bit preferred, 16-bit with TPDF dither
• LUFS normalized per category • True-peak ceiling −0.3 dBTP

Tested on: MPC Live II, MPC III, MPC One+, MPC Control (your hardware fleet)
