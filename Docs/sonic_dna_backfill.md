## Heuristic System

Values were generated by `Tools/add_missing_dna.py` using a three-layer heuristic:

### Layer 1 — Mood Baseline

Each of the 6 moods has a characteristic DNA signature:

| Mood | brightness | warmth | aggression | movement | density | space |
|------|-----------|--------|-----------|----------|---------|-------|
| Foundation | 0.4 | 0.6 | 0.2 | 0.2 | 0.3 | 0.6 |
| Atmosphere | 0.3 | 0.5 | 0.1 | 0.5 | 0.5 | 0.7 |
| Prism | 0.8 | 0.3 | 0.4 | 0.6 | 0.7 | 0.4 |
| Flux | 0.5 | 0.3 | 0.6 | 0.8 | 0.8 | 0.3 |
| Entangled | 0.5 | 0.4 | 0.5 | 0.7 | 0.9 | 0.5 |
| Aether | 0.4 | 0.2 | 0.1 | 0.4 | 0.6 | 0.9 |

### Layer 2 — Engine Identity Nudge

Each engine applies a characteristic delta to the mood baseline. For XOwlfish
(subharmonic bass engine):

| Dimension | Delta |
|-----------|-------|
| warmth | +0.15 |
| space | +0.10 |
| brightness | -0.10 |
| aggression | -0.05 |

All nudges are averaged across engines if a preset uses multiple engines.

### Layer 3 — Name and Tag Keywords

Keywords found in the preset name, description, or tags apply additional ±0.20 shifts:

| Keywords | Effect |
|----------|--------|
| bright, crystal, shimmer, photon, bioluminescent | brightness +0.20 |
| warm, bass, sub, body, heavy, abyssal, floor, trench | warmth +0.20 |
| dark, midnight, shadow, abyss, void, deep | brightness -0.20 |
| aggressive, growl, distorted, predator, hunter, frenzy | aggression +0.20 |
| swarm, granular, chaotic, scatter, storm, frenzy | movement +0.20 |
| drone, static, still, meditative, solitary, infinite | movement -0.20 |
| reverb, vast, deep, space, open, migration, drift | space +0.20 |
| layered, ecosystem, coupling, harmonic, complex | density +0.20 |
| sparse, solo, lone, solitary, minimal | density -0.20 |
| percussive, snap, kick, explosive, transient | aggression +0.15, movement +0.10 |
| fragment, scatter, armor | movement +0.15, density +0.10 |

Final values are clamped to [0.0, 1.0].

## Important Note — Values Are Estimates

These DNA values were generated by heuristic, not by acoustic analysis of the actual
audio output. They are **directionally correct** — a dark sub-bass Foundation preset
correctly receives low brightness and high warmth — but the exact magnitudes have not
been verified by listening.

**Future manual curation should:**

1. Play each XOwlfish preset and compare its subjective character to the DNA values
2. Adjust any dimension that feels misrepresented (e.g. if `space` feels too high/low)
3. Rerun `python3 Tools/audit_sonic_dna.py` after any changes to verify coverage is maintained

The XOwlfish engine currently shows two coverage gaps in the audit report:
- `warmth` min > 0.30 (all XOwlfish presets are inherently warm — structural identity limit)
- `space` max near 1.00 (XOwlfish is a deep, resonant engine — high space is expected)

These are engine identity characteristics, not backfill errors.
