## How to Use This Index

Every concept brief follows Phase 0 format: Identity, Thesis, Sound family, Water column placement, Gallery code, accent color, parameter prefix. Documents marked **(retrospective)** were written after the instrument was built. Documents marked **(draft)** are pre-approval ideation. All others are approved and ready for Phase 1 architecture.
