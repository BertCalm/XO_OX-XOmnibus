# XO_OX Sonic Identity System

**Status**: R&D Spec | March 2026
**Purpose**: Brand standards for sonic character — how XO_OX packs are positioned, described, and recognized.

---

## 1. The Core Axes

Every XO_OX sound lives on two axes:

**feliX ↔ Oscar** (horizontal)
feliX = bright, clinical, sharp, spectral clarity
Oscar = warm, organic, deep, harmonic density

**Surface ↔ Depth** (vertical)
Surface = immediate, present, attack, transient-forward
Depth = sustained, environmental, textural, immersive

### The Four Quadrants

| | Surface | Depth |
|---|---|---|
| **feliX** | Cutting transients, surgical presence, snap and crack | Clinical pads, crystalline atmospheres, spectral shimmer |
| **Oscar** | Warm hits, analog punch, character drums | Environmental texture, sub-drones, vast organic rooms |

**Engine examples:**
- **Surface feliX** — ONSET (drum snap, transient clarity), ODDFELIX (percussive brightness)
- **Depth feliX** — ORACLE (spectral synthesis, long crystalline tails), ODYSSEY (wavetable shimmer, sustained clinical motion)
- **Surface Oscar** — OBLONG (analog character hits, warm low-mid punch), OBESE (fat transients, weighted presence)
- **Depth Oscar** — OPAL (granular warmth, slow environmental immersion), OVERDUB (dub reverb depth, tape warmth)

---

## 2. Sonic Signature Cards

Every preset ships with a 3-line Sonic Signature Card embedded in metadata and displayed in OXPort/the web catalog.

**Format:**

```
Line 1: [Axis Position] — e.g., "Deep Oscar, lightly feliX"
Line 2: [Plain language character] — e.g., "Warm granular shimmer with slow attack and vast room"
Line 3: [Use case] — e.g., "Atmospheric pad / hip-hop intro / cinematic underscoring"
```

**Rules:**
- Line 1 must use only axis vocabulary (feliX, Oscar, Surface, Depth, and modifiers: lightly/heavily/balanced)
- Line 2: no synthesis jargon ("granular" is acceptable shorthand; "FM ratio 3:2" is not)
- Line 3: production context first, genre second — "lead hook" before "EDM drop"

**Example — OPAL preset "Tide Fracture":**
```
Deep Oscar, lightly feliX
Slow granular breath with warm harmonic wash and distant shimmer
Cinematic texture / ambient intro / hip-hop sample flip
```

## 4. Fleet Position Map

**Surface feliX quadrant** (cutting, clinical, immediate):
ONSET, ODDFELIX
Character: transient precision, rhythmic definition

**Depth feliX quadrant** (sustained, spectral, bright):
ORACLE, ODYSSEY, ORPHICA, OHM
Character: long crystalline tails, spectral motion, cerebral texture

**Surface Oscar quadrant** (warm, punchy, present):
OBLONG, OBESE, ONSET (low voices), OTTONI, OLE
Character: analog weight, mid-body warmth, immediate impact

**Depth Oscar quadrant** (environmental, immersive, organic):
OPAL, OVERDUB, OVERLAP, OUTWIT
Character: vast rooms, granular breath, textural depth

**Fleet center of gravity**: **mid-Oscar, leaning Depth.** XO_OX is not a clinical tool library — it makes sounds that have weight and place. The fleet earns feliX contrast through selective engines (ONSET, ORACLE) rather than leading with brightness.

---

## 5. Pack Identity Rules

**Rule 1: Packs occupy a region, not a scatter plot.**
A coherent pack stays within one quadrant or spans one adjacent edge (e.g., Surface Oscar to Surface feliX). A pack that spans all four quadrants is a starter kit, not an identity pack. Name accordingly — "XO_OX Essentials" is a valid scatter; "Tide Tables" should not be.

**Rule 2: The engine mix should justify the pack name.**
If the pack is called "Weight," its engines should be Oscar-dominant. If it's called "Splice," feliX-dominant. Mismatches between name and axis position break brand trust faster than any individual bad preset.

**Rule 3: Every pack needs a declared center and a declared contrast.**
State one primary quadrant (the "home") and one secondary quadrant (the "tension"). Home delivers the promise; tension provides the release. A pack with no tension is one-dimensional. A pack with no home is incoherent.
