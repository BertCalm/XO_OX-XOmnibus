# OMBRE XPN Pack — Design R&D

**Engine**: OMBRE | **Prefix**: `ombre_` | **Accent**: Shadow Mauve `#7B6B8A`
**Date**: 2026-03-16 | **Status**: Concept

---

## 1. Pack Identity

**Pack Name**: *The Space Between Remembering*

**Tagline**: "What you almost forgot. What you can't quite hold."

**DNA Position**: OMBRE sits at the center of the Sonic DNA map — warmth 0.6, density 0.6, movement medium. It is not extreme in any direction. This is the point. OMBRE is the in-between engine: not fully luminous, not fully buried, not still, not turbulent. Every preset occupies a position along a gradient rather than a pole. This makes OMBRE the XO_OX engine most suited to emotional ambiguity — the feeling of a memory that is real but no longer sharp.

**Tier Recommendation**: Tier 2 (Signature). OMBRE demands enough craft in velocity mapping and dual-voice balance that it rewards experienced players. A free sampler (2 pads, one from Row A, one from Row D) is appropriate for discovery.

---

## 2. OMBRE's Dual Narrative

OMBRE runs two simultaneous voices at all times. Voice 1 is **Memory**: crystalline, high-frequency detail intact, attack precise — but decaying. Voice 2 is **Forgetting**: warm, saturated, smeared attack — but sustaining, filling the space the memory leaves behind.

The tension is that neither voice wins. A preset does not resolve into clarity or into warmth. It holds both states in superposition: something being remembered and something being lost at the same time. The `ombre_` parameter space controls the ratio between these voices — how much of the signal belongs to each narrative — as well as the rate at which Memory fades and Forgetting rises.

This creates presets that feel like they are **remembering themselves**: the initial transient is sharp but leaves a warm residue, and the sustain is soft but carries the ghost of what was distinct at the attack. No other XO_OX engine produces this simultaneous forward/backward temporal sensation.

## 3. Pack Structure — 16 Pads

### Row A (Pads 1–4): Clear Memory
Memory voice dominant. Full luminosity, intact high-frequency content, tight decay. Forgetting voice present but barely audible — a faint warmth at the tail.

- **Pad 1** `Photographic` — snapshot-sharp attack, nearly zero sustain bleed
- **Pad 2** `First Light` — slightly longer tail, Memory voice still 80% dominant
- **Pad 3** `Vivid Detail` — Memory at full saturation; Forgetting voice adds only the softest halo
- **Pad 4** `Total Recall` — showcase pad: every parameter of the Memory voice pushed to maximum expression

### Row B (Pads 5–8): Fading Memory
Transition zone. Memory still recognizable but degrading. Warmth rising. Attack rounding. The moment a thought starts to slip.

- **Pad 5** `Soft Edges` — first detectable blur on the transient
- **Pad 6** `Second Reading` — memory read twice; the second reading is warmer and less precise
- **Pad 7** `Half-Life` — exact balance point where sharpness and warmth are equal in energy (not equal in perception — warmth always feels larger)
- **Pad 8** `The Drift` — Memory voice audibly in retreat; Forgetting voice taking the low-mids

### Row C (Pads 9–12): Forgetting
Forgetting voice dominant. Warmth and blur in control. Memory present only as faint high-frequency shimmer — not gone, just distant.

- **Pad 9** `Warm Static` — primarily Forgetting voice with a thin crystal layer above
- **Pad 10** `Last Syllable` — the moment after a word is spoken and the phoneme is dissolving
- **Pad 11** `Deep Amber` — warm, saturated, slow; Memory voice reduced to a shimmer
- **Pad 12** `Almost Gone` — Forgetting dominant at 85%; the shadow of Memory barely detectable

### Row D (Pads 13–16): The Gradient
Full OMBRE character. Both voices balanced and expressive. These are the most playable pads — the dual narrative is audibly present and controllable via velocity and Q-Links.

- **Pad 13** `Shadow Mauve` — named for the accent color; the two voices woven equally
- **Pad 14** `Gradient State` — slow movement between Memory and Forgetting across the note's life
- **Pad 15** `The Long Return` — extended decay that cycles: clarity → warmth → clarity → warmth
- **Pad 16** `Persistent Echo` — showcase pad: both narratives at maximum expression, full OMBRE arc in a single note

## 4. Preset Design Philosophy

OMBRE presets should feel like **time passing inside a single note**. The attack is one moment in time; the release is a different moment, further from the source. This means:

- **Velocity = memory strength**: soft velocity triggers full Memory voice (quiet = full recall — the thought is close, intact, clear). Loud velocity triggers full Forgetting voice (loud = the thought has been played many times and worn smooth — warmth replaces sharpness). This is counterintuitive but emotionally correct.
- **Long release times** are preferred throughout the pack. OMBRE should never feel abrupt.
- **No hard transients in Row C or D** — Forgetting cannot have a sharp attack without breaking the narrative.
- **Subtle pitch drift** (slow, barely perceptible) on Forgetting voice — memory is not pitch-stable.

## 5. MPCe Quad-Corner

| Corner | Character | Voice Balance | Mood |
|--------|-----------|---------------|------|
| **NW** | Sharp Memory / feliX | Memory 90%, Forgetting 10% | Precise, present, clear |
| **NE** | Warm Memory / Balanced | Memory 60%, Forgetting 40% | Nostalgic but intact |
| **SW** | Forgotten / Oscar | Forgetting 90%, Memory 10% | Buried, soft, distant |
| **SE** | Gradient / Blend | 50/50, both voices active | Full OMBRE character |

The SE corner (Gradient) is the most expressive Q-Link destination — both voices respond to the performance and the player can feel the tension between them.

## 6. Pack DNA Arc

The pack traces a single arc across all 16 pads:

| Row | Brightness | Warmth | Memory % | Forgetting % |
|-----|-----------|--------|----------|--------------|
| A (1–4) | High | Low | 85–95% | 5–15% |
| B (5–8) | Medium-High | Medium | 55–70% | 30–45% |
| C (9–12) | Low-Medium | High | 15–30% | 70–85% |
| D (13–16) | Variable | Variable | 45–55% | 45–55% |

Row D breaks the arc intentionally — it is not a continuation into full darkness but a return to the center. The pack ends in balance, not resolution. This is the OMBRE doctrine: the gradient never finishes.

## 7. Coupling Chemistry

**OMBRE × ODYSSEY** — Drift amplifies fade. ODYSSEY's pitch drift pairs with OMBRE's timbral drift; together they produce the sensation of a memory moving away in two dimensions simultaneously — pitch and texture both receding. Recommended coupling: OMBRE Memory voice → ODYSSEY drift input. The result is a sound that remembers being in tune.

**OMBRE × OVERDUB** — Tape decay mirrors memory decay. OVERDUB's tape saturation is mechanically similar to OMBRE's Forgetting voice (both warm, both corrupting). Layering them doubles the warmth without losing the Memory voice from OMBRE — OVERDUB fills the low-mid space that Memory vacates, creating a richer Forgetting. Recommended coupling: parallel routing, OMBRE primary, OVERDUB as texture layer.

**OMBRE × ORACLE** — The deepest pairing. ORACLE concerns what is foreseen; OMBRE concerns what is remembered. Together they produce a temporal surround: one engine looking forward into uncertainty, one looking backward into dissolution. What ORACLE predicts, OMBRE forgets. What OMBRE remembers, ORACLE cannot foresee. Recommended coupling: ORACLE on attack transient, OMBRE on sustain/release — the note begins as prophecy and ends as memory.
