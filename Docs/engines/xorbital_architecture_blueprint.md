# XOrbital — Architecture Blueprint (Phase 1)

**Date:** March 2026
**Status:** Phase 1 — Architecture designed, pending scaffold
**Engine location:** `Source/Engines/Orbital/OrbitalEngine.h` (native XOceanus engine, self-contained)

## 1. Architecture Pattern

**Self-contained inline header** — all DSP lives in `OrbitalEngine.h` with embedded helper structs at the top (following ONSET's pattern). No external library delegation (unlike OPAL which wraps XOpal). This keeps the engine portable, testable, and eliminates include-path complexity.

**Estimated header size:** ~1200–1500 lines (comparable to ONSET's 1041 lines, larger due to 64-partial state and spectral profile data).

### Helper Struct Hierarchy

```
SpectralProfile          — 64-point harmonic amplitude template (static data)
FormantFilter            — 64-point spectral envelope with vowel presets
OrbitalGroupEnvelope     — Per-group AD envelope with shared S/R
OrbitalVoice             — 64-partial voice: phases, amplitudes, group envs, FM state
OrbitalEngine            — SynthEngine impl: MIDI, coupling, params, render
```

## 3. Spectral Profiles — Static Data

Each profile is a `std::array<float, 64>` of normalized harmonic amplitudes. Two profiles (A and B) morph continuously via `orb_morph` parameter.

### 3.1 Profile Presets (8 built-in)

```cpp
struct SpectralProfile
{
    const char* name;
    std::array<float, 64> amplitudes;  // amplitudes[k] = amplitude of partial k+1
};
```

| Index | Name | Formula / Character |
|-------|------|-------------------|
| 0 | Sawtooth | `amp[k] = 1.0 / (k+1)` — all harmonics, natural rolloff |
| 1 | Square | `amp[k] = (k % 2 == 0) ? 1.0 / (k+1) : 0.0` — odd harmonics only |
| 2 | Triangle | `amp[k] = (k % 2 == 0) ? 1.0 / ((k+1)*(k+1)) : 0.0` — odd harmonics, steep rolloff |
| 3 | Bell | `amp[k] = exp(-0.15 * k) * (1.0 + 0.3 * sin(k * 0.7))` — decaying with spectral ripple |
| 4 | Organ | `amp[k] = drawbar(k)` — emulates 9-drawbar registration: partials 1,2,3,4,5,6,8 emphasized |
| 5 | Vocal | `amp[k] = formantEnvelope(k, 'ah')` — "ah" vowel spectral shape as base profile |
| 6 | Glass | `amp[k] = (k < 8) ? 0.1 : exp(-0.05 * (k - 8))` — weak fundamental, strong upper partials |
| 7 | Custom | User-definable — defaults to flat (all equal at 1.0/64) |

### 3.2 Profile Data Generation

Profiles are **compile-time constants** — generated by `constexpr` functions or static initializer lists. No runtime computation.

```cpp
// Example: Sawtooth profile generation
static constexpr SpectralProfile makeSawtooth()
{
    SpectralProfile p { "Sawtooth", {} };
    for (int k = 0; k < 64; ++k)
        p.amplitudes[k] = 1.0f / static_cast<float>(k + 1);
    return p;
}

static constexpr std::array<SpectralProfile, 8> kProfiles = {
    makeSawtooth(),
    makeSquare(),
    makeTriangle(),
    makeBell(),
    makeOrgan(),
    makeVocal(),
    makeGlass(),
    makeCustom()
};
```

### 3.3 Organ Drawbar Registration

The Organ profile emulates a tonewheel organ's harmonic structure. Drawbar contributions map to specific partials:

| Drawbar | Footage | Partial Index | Relative Level |
|---------|---------|---------------|----------------|
| 1 | 16' | 0 (sub) | 0.6 |
| 2 | 5⅓' | 2 (3rd) | 0.4 |
| 3 | 8' | 1 (fundamental) | 1.0 |
| 4 | 4' | 3 (4th) | 0.8 |
| 5 | 2⅔' | 5 (6th) | 0.5 |
| 6 | 2' | 7 (8th) | 0.7 |
| 7 | 1⅗' | 9 (10th) | 0.3 |
| 8 | 1⅓' | 11 (12th) | 0.2 |
| 9 | 1' | 15 (16th) | 0.15 |

All other partials in the Organ profile are set to 0.0. This gives a warm, full-bodied sound without upper shimmer.

## 4. Formant Filter — Spectral Envelope

The formant filter is NOT a time-domain filter — it's a 64-point amplitude multiplier array applied per-partial. Computationally free (the multiply is already part of the per-partial amplitude computation).

### 4.1 Data Structure

```cpp
struct FormantFilter
{
    std::array<float, 64> envelope;  // per-partial amplitude multiplier [0..1]

    // Apply spectral envelope to a partial's amplitude
    // Returns: amp * envelope[partialIndex]
    float apply(int partialIndex, float amp) const
    {
        return amp * envelope[partialIndex];
    }

    // Build a vowel formant for a given fundamental frequency
    void buildVowel(int vowelIndex, float fundamentalHz, float sampleRate);

    // Shift the entire formant up/down by semitones
    void shift(float semitones, float fundamentalHz);

    // Build from spectral tilt + odd/even balance
    void buildFromControls(float tilt, float oddEven);
};
```

### 4.2 Vowel Formant Data

Vowel formants are defined by 3 resonant peaks (F1, F2, F3) in Hz. At `buildVowel()` time, these are mapped to the nearest partial indices for the current fundamental frequency.

| Vowel | F1 (Hz) | F2 (Hz) | F3 (Hz) | Character |
|-------|---------|---------|---------|-----------|
| A ("ah") | 730 | 1090 | 2440 | Open, warm |
| E ("eh") | 530 | 1840 | 2480 | Mid, nasal |
| I ("ee") | 270 | 2290 | 3010 | Bright, thin |
| O ("oh") | 570 | 840 | 2410 | Round, dark |
| U ("oo") | 300 | 870 | 2240 | Closed, deep |

**Formant envelope computation** (called once per block when formant params change):

```
For each partial k (0..63):
    freq_k = (k + 1) * fundamentalHz
    gain_k = 0.0

    For each formant peak (F1, F2, F3):
        // Gaussian resonance peak centered at formant frequency
        distance = (freq_k - Fn) / bandwidth_n
        gain_k += peakAmp_n * exp(-0.5 * distance * distance)

    envelope[k] = clamp(gain_k, 0.0, 1.0)
```

**Bandwidth:** Each formant peak has bandwidth of ~100–150 Hz (wider for lower formants). Stored as compile-time constants per vowel.

### 4.3 Spectral Tilt + Odd/Even

When no vowel is selected (formantShape = Off), the formant filter is built from two simpler controls:

- **Spectral tilt** (`orb_brightness`, 0–1): Controls rolloff rate of upper partials
  - `envelope[k] = pow(1.0 / (k + 1), (1.0 - brightness) * 2.0)`
  - brightness=0 → steep rolloff (dark). brightness=1 → flat (all harmonics equal).

- **Odd/Even balance** (`orb_oddEven`, 0–1):
  - 0.0 = odd-only (square-ish): even partials multiplied by 0.0
  - 0.5 = equal: all partials at 1.0×
  - 1.0 = even-emphasized: odd partials (except fundamental) multiplied by 0.3

### 4.4 Formant Shift

`orb_formantShift` (±24 semitones) transposes the entire spectral envelope by shifting which partials are boosted:

```
shiftedIndex = round(k * pow(2.0, -semitones / 12.0))
envelope[k] = sourceEnvelope[clamp(shiftedIndex, 0, 63)]
```

Recomputed once per block when the parameter changes. Uses a dirty flag to avoid redundant computation.

## 5. Voice Architecture

### 5.1 OrbitalVoice Struct

```cpp
struct OrbitalVoice
{
    bool active = false;
    int noteNumber = -1;
    float velocity = 0.0f;
    uint64_t startTime = 0;        // for LRU voice stealing

    // --- Per-partial state (64 partials) ---
    double phase[64] = {};          // double precision — prevents drift at low freqs
    float  phaseIncF[64] = {};      // float cache of phase increment (updated at noteOn)
    float  inharmRatio[64] = {};    // cached inharmonicity ratios (computed at noteOn)

    // --- FM state (when fmIndex > 0) ---
    double fmPhase[64] = {};        // FM modulator phase per partial
    float  fmPhaseInc[64] = {};     // FM modulator phase increment

    // --- Group envelopes (4 groups) ---
    // Group 1: partials 0-7,  Group 2: 8-15
    // Group 3: 16-31,         Group 4: 32-63
    float groupEnvLevel[4] = {};    // current envelope level per group
    float groupAttackCoeff[4] = {}; // per-sample attack coefficient
    float groupDecayCoeff[4] = {};  // per-sample decay coefficient
    enum class GroupEnvStage { Attack, Decay, Sustain, Release, Off };
    GroupEnvStage groupStage[4] = {};

    // --- Global ADSR ---
    float envLevel = 0.0f;
    float envAttackCoeff = 0.0f;
    float envDecayCoeff = 0.0f;
    float envSustain = 0.0f;
    float envReleaseCoeff = 0.0f;
    enum class EnvStage { Attack, Decay, Sustain, Release, Off };
    EnvStage envStage = EnvStage::Off;

    // --- Voice stealing crossfade ---
    float fadeOutLevel = 0.0f;

    // --- Fundamental ---
    float fundamentalHz = 0.0f;

    // --- Per-partial pan coefficients (computed at noteOn) ---
    float panL[64] = {};
    float panR[64] = {};
};
```

### 5.2 Memory Layout

Per voice: `64 × 8` (phase, double) + `64 × 4 × 6` (float arrays) + envelope state + overhead ≈ **2.2 KB per voice**. Total for 6 voices: **~13.2 KB**. Fits comfortably in L1 cache.

### 5.3 noteOn()

All expensive computation happens here — never in the per-sample loop.

```cpp
void triggerVoice(OrbitalVoice& v, int note, float vel, float sampleRate,
                  float inharm, float fmRatio, float stereoSpread,
                  float attackS, float decayS, float sustain, float releaseS,
                  float grpAtk1, float grpDec1, float grpAtk2, float grpDec2,
                  float grpAtk3, float grpDec3, float grpAtk4, float grpDec4)
{
    v.active = true;
    v.noteNumber = note;
    v.velocity = vel;
    v.startTime = voiceCounter++;
    v.fundamentalHz = midiToFreq(note);

    // --- Inharmonicity ratios (cached — no sqrt in render loop) ---
    // Piano-string model: ratio[k] = sqrt(1 + B * k^2)
    // B = inharm parameter mapped to [0, 0.001] (subtle) range
    float B = inharm * 0.001f;
    for (int k = 0; k < 64; ++k)
    {
        float kf = static_cast<float>(k + 1);
        v.inharmRatio[k] = std::sqrt(1.0f + B * kf * kf);  // sqrt OK here — noteOn only
    }

    // --- Phase increments ---
    double invSr = 1.0 / sampleRate;
    constexpr double twoPi = 6.283185307179586;
    for (int k = 0; k < 64; ++k)
    {
        double partialFreq = static_cast<double>(v.fundamentalHz)
                           * static_cast<double>(k + 1)
                           * static_cast<double>(v.inharmRatio[k]);
        v.phaseIncF[k] = static_cast<float>(partialFreq * twoPi * invSr);
        v.phase[k] = 0.0;  // reset phase for clean attack

        // FM modulator phase increments
        v.fmPhaseInc[k] = static_cast<float>(partialFreq * fmRatio * twoPi * invSr);
        v.fmPhase[k] = 0.0;
    }

    // --- Stereo spread: per-partial pan ---
    // Partials spread outward from center as index increases
    for (int k = 0; k < 64; ++k)
    {
        float spread = stereoSpread * static_cast<float>(k) / 63.0f;
        float angle = spread * 0.5f;  // max ±0.5 pan
        // Alternate L/R: even partials go left, odd go right
        if (k % 2 == 1) angle = -angle;
        // Convert to pan coefficients (constant-power)
        constexpr float halfPi = 1.5707963f;
        float panPos = 0.5f + angle;  // [0..1]
        v.panL[k] = std::cos(panPos * halfPi);  // cos/sin OK — noteOn only
        v.panR[k] = std::sin(panPos * halfPi);
    }

    // --- Global ADSR coefficients ---
    v.envLevel = 0.0f;
    v.envStage = OrbitalVoice::EnvStage::Attack;
    v.envAttackCoeff = 1.0f / (std::max(0.001f, attackS) * static_cast<float>(sampleRate));
    v.envDecayCoeff = 1.0f / (std::max(0.05f, decayS) * static_cast<float>(sampleRate));
    v.envSustain = sustain;
    v.envReleaseCoeff = 1.0f / (std::max(0.05f, releaseS) * static_cast<float>(sampleRate));

    // --- Group envelope coefficients ---
    float grpAttacks[4] = { grpAtk1, grpAtk2, grpAtk3, grpAtk4 };
    float grpDecays[4]  = { grpDec1, grpDec2, grpDec3, grpDec4 };
    for (int g = 0; g < 4; ++g)
    {
        v.groupEnvLevel[g] = 0.0f;
        v.groupStage[g] = OrbitalVoice::GroupEnvStage::Attack;
        v.groupAttackCoeff[g] = 1.0f / (std::max(0.001f, grpAttacks[g])
                                        * static_cast<float>(sampleRate));
        v.groupDecayCoeff[g] = 1.0f / (std::max(0.01f, grpDecays[g])
                                       * static_cast<float>(sampleRate));
    }

    v.fadeOutLevel = 0.0f;
}
```

### 5.4 Group → Partial Index Mapping

| Group | Partial Indices | Count | Character |
|-------|----------------|-------|-----------|
| 0 | 0–7 | 8 | Fundamental region — body, warmth |
| 1 | 8–15 | 8 | Mid harmonics — character, presence |
| 2 | 16–31 | 16 | Upper harmonics — brightness, edge |
| 3 | 32–63 | 32 | Air harmonics — shimmer, sparkle |

```cpp
static constexpr int groupForPartial(int k)
{
    if (k < 8)  return 0;
    if (k < 16) return 1;
    if (k < 32) return 2;
    return 3;
}
```

## 6. Render Loop

### 6.1 renderBlock() Top-Level Flow

```
1. ScopedNoDenormals
2. buffer.clear()
3. ParamSnapshot — read all parameters once
4. Recompute formant envelope if dirty (vowel changed, shift changed, tilt changed)
5. Process MIDI → noteOn/noteOff/allNotesOff
6. Apply macros to param snapshot
7. Consume coupling accumulators, reset them
8. Per-sample loop:
   a. For each active voice:
      i.   Tick global ADSR
      ii.  Tick 4 group envelopes
      iii. For each partial k (0..63):
           - amp = lerp(profileA[k], profileB[k], morph) × formantEnv[k] × groupEnv[group]
           - If amp < threshold (0.001), skip this partial
           - If FM active: phase += phaseInc + fmIndex × fastSin(fmPhase)
           - Else: phase += phaseInc
           - sample = amp × fastSin(phase)
           - Accumulate to voiceL/R via panL[k], panR[k]
      iv.  Apply voice-stealing crossfade
      v.   Scale by velocity and global ADSR
      vi.  Accumulate to output buffer
   b. Write to outputCacheL/R for coupling
9. Post filter (CytomicSVF on summed stereo output, coeffs cached per block)
10. Saturator character stage
11. Master volume
12. Cache last samples for coupling output
```

### 6.2 Per-Sample Inner Loop (Critical Path)

This is the performance-critical code. Every operation has been analyzed for cost.

```cpp
// Inside the per-sample loop, per-voice:
float voiceL = 0.0f, voiceR = 0.0f;

for (int k = 0; k < 64; ++k)
{
    // --- Amplitude: profile morph × formant × group envelope ---
    int g = groupForPartial(k);
    float amp = lerp(profileA[k], profileB[k], morphPos)
              * formantEnv[k]
              * v.groupEnvLevel[g];

    // Early-out: skip silent partials (saves ~40% on average)
    if (amp < 0.001f) continue;

    float sample;
    if (fmIndex > 0.001f)
    {
        // FM path: modulate phase increment
        float fmMod = fmIndex * fastSin(static_cast<float>(v.fmPhase[k]));
        v.phase[k] += static_cast<double>(v.phaseIncF[k] + fmMod * v.phaseIncF[k]);
        v.fmPhase[k] += static_cast<double>(v.fmPhaseInc[k]);
        sample = amp * fastSin(static_cast<float>(v.phase[k]));
    }
    else
    {
        // Clean path: simple phase accumulation
        v.phase[k] += static_cast<double>(v.phaseIncF[k]);
        sample = amp * fastSin(static_cast<float>(v.phase[k]));
    }

    // Stereo accumulation (pan coefficients precomputed at noteOn)
    voiceL += sample * v.panL[k];
    voiceR += sample * v.panR[k];
}

// Denormal flush on phase accumulators (every N samples, not every sample)
// See §6.5 for denormal strategy
```

### 6.3 Partial Skip Optimization

The early-out `if (amp < 0.001f) continue` is crucial. In practice:

| Scenario | Active Partials | Skipped | Savings |
|----------|----------------|---------|---------|
| Sawtooth, brightness=1.0 | ~64 | ~0 | 0% |
| Sawtooth, brightness=0.5 | ~32 | ~32 | 50% |
| Organ profile | ~9 | ~55 | 86% |
| Square + formant "ee" | ~12 | ~52 | 81% |
| Typical preset (avg) | ~25 | ~39 | 61% |

This means typical CPU usage is **6–10%**, not the theoretical maximum of 15%.

### 6.4 CPU Budget (Revised)

| Component | Ops/sample (worst case) | Ops/sample (typical) |
|-----------|------------------------|---------------------|
| 64 × fastSin (6 voices) | 384 × ~14 = 5,376 | ~150 × ~14 = 2,100 |
| 64 × FM fastSin (6 voices) | 384 × ~14 = 5,376 | 0 (FM off in most presets) |
| Phase updates (double add) | 384 × ~3 = 1,152 | ~150 × ~3 = 450 |
| Amplitude computation | 384 × ~6 = 2,304 | ~150 × ~6 = 900 |
| Pan accumulation | 384 × ~4 = 1,536 | ~150 × ~4 = 600 |
| ADSR + group envs (6 voices) | 6 × ~20 = 120 | 120 |
| Post filter (1× CytomicSVF) | ~50 | ~50 |
| Saturator | ~30 | ~30 |
| **Total** | **~15,944** | **~4,250** |
| **% of 3GHz single core** | **~23%** | **~6%** |

Worst case exceeds the 20% target slightly but only with FM on all 64 partials across 6 voices — an extreme edge case. Typical usage is well under 15%.

### 6.5 Denormal Strategy

Phase accumulators use `double` precision and accumulate continuously. Denormals can appear when:
- A voice is very quiet (amp ≈ 0)
- Phase values grow very large (loss of precision, not denormals per se)

Strategy:
1. **`juce::ScopedNoDenormals`** at the top of renderBlock() — sets FTZ/DAZ flags for the entire block
2. **Phase wrapping** every 1024 samples: `if (phase[k] > twoPi * 1e6) phase[k] -= twoPi * floor(phase[k] / twoPi)` — prevents precision loss
3. **`flushDenormal()`** on the post-filter and saturator state variables (CytomicSVF already does this internally)
4. **`flushDenormal()`** on group envelope levels when they drop below threshold

```cpp
// Phase wrapping — called every 1024 samples (not every sample)
if ((sampleCounter & 0x3FF) == 0)
{
    constexpr double twoPi = 6.283185307179586;
    constexpr double wrapThreshold = twoPi * 1e6;
    for (auto& v : voices)
    {
        if (!v.active) continue;
        for (int k = 0; k < 64; ++k)
        {
            if (v.phase[k] > wrapThreshold)
                v.phase[k] -= twoPi * std::floor(v.phase[k] / twoPi);
            if (fmIndex > 0.001f && v.fmPhase[k] > wrapThreshold)
                v.fmPhase[k] -= twoPi * std::floor(v.fmPhase[k] / twoPi);
        }
    }
}
```

## 7. Envelope System

### 7.1 Global ADSR

Standard ADSR applied to the entire voice output. Per-sample linear segments.

```cpp
// Per-sample tick (inside render loop)
switch (v.envStage)
{
    case EnvStage::Attack:
        v.envLevel += v.envAttackCoeff;
        if (v.envLevel >= 1.0f) { v.envLevel = 1.0f; v.envStage = EnvStage::Decay; }
        break;
    case EnvStage::Decay:
        v.envLevel -= v.envDecayCoeff;
        if (v.envLevel <= v.envSustain) { v.envLevel = v.envSustain; v.envStage = EnvStage::Sustain; }
        break;
    case EnvStage::Sustain:
        // Hold at sustain level until noteOff
        break;
    case EnvStage::Release:
        v.envLevel -= v.envReleaseCoeff;
        if (v.envLevel <= 0.0f) { v.envLevel = 0.0f; v.envStage = EnvStage::Off; v.active = false; }
        break;
    case EnvStage::Off:
        break;
}
```

### 7.2 Group Envelopes (AD with shared S/R)

Each of the 4 groups has independent Attack and Decay times. Sustain and Release are shared with the global ADSR. The group envelope multiplies the per-partial amplitude.

The group envelope runs AD (Attack → Decay → hold at 1.0). When the global ADSR enters Release, all group envelopes follow the global release curve.

```cpp
// Per-sample group envelope tick
for (int g = 0; g < 4; ++g)
{
    switch (v.groupStage[g])
    {
        case GroupEnvStage::Attack:
            v.groupEnvLevel[g] += v.groupAttackCoeff[g];
            if (v.groupEnvLevel[g] >= 1.0f)
            {
                v.groupEnvLevel[g] = 1.0f;
                v.groupStage[g] = GroupEnvStage::Decay;
            }
            break;
        case GroupEnvStage::Decay:
            v.groupEnvLevel[g] -= v.groupDecayCoeff[g];
            if (v.groupEnvLevel[g] <= 0.3f)  // decay floor (don't go silent)
            {
                v.groupEnvLevel[g] = 0.3f;
                v.groupStage[g] = GroupEnvStage::Sustain;
            }
            break;
        case GroupEnvStage::Sustain:
            // Hold — follows global ADSR sustain
            break;
        case GroupEnvStage::Release:
            // Follows global release — handled by global ADSR multiplying voice output
            break;
        case GroupEnvStage::Off:
            break;
    }
}
```

**Musical effect example:** Upper partials (Group 3+4) with fast attack and fast decay = bell attack that fades, while lower partials (Group 1) with slow attack = warm body that swells in. One preset parameter set creates a sound that attacks bright and settles warm.

## 8. Coupling Interface Implementation

### 8.1 Coupling Accumulators

```cpp
// Member variables — reset each block in renderBlock()
float externalFilterMod  = 0.0f;   // Hz offset on post-filter cutoff
float externalMorphMod   = 0.0f;   // offset on morph position [0..1]
float externalPitchMod   = 0.0f;   // semitone offset on fundamental
float externalFormantMod = 0.0f;   // semitone offset on formant shift
float externalFmMod      = 0.0f;   // additive FM index from external audio
float externalDecayMod   = 0.0f;   // multiplier on group decay times
float externalBlendMod   = 0.0f;   // offset on harmonic group enable

// Tight coupling buffers (per-sample)
juce::AudioBuffer<float> couplingAudioBuf;   // AudioToWavetable input
juce::AudioBuffer<float> couplingRingBuf;    // AudioToRing input
```

### 8.2 applyCouplingInput() Switch

```cpp
void applyCouplingInput(CouplingType type, float amount,
                        const float* sourceBuffer, int numSamples) override
{
    switch (type)
    {
        case CouplingType::AudioToWavetable:
        {
            // The killer coupling: source audio reshapes partial amplitudes.
            // Buffer stored for per-sample processing in renderBlock.
            if (sourceBuffer != nullptr)
            {
                float* cL = couplingAudioBuf.getWritePointer(0);
                for (int n = 0; n < numSamples; ++n)
                    cL[n] = sourceBuffer[n] * amount;
            }
            break;
        }

        case CouplingType::AmpToFilter:
            // Source amplitude → post-filter cutoff offset (±8000 Hz)
            externalFilterMod = amount * 8000.0f;
            break;

        case CouplingType::EnvToMorph:
            // Source envelope → profile A↔B morph position offset
            externalMorphMod = amount * 0.5f;
            break;

        case CouplingType::LFOToPitch:
            // Source LFO → fundamental pitch offset (±2 semitones)
            externalPitchMod += amount * 2.0f;
            break;

        case CouplingType::PitchToPitch:
            // Source pitch → fundamental pitch offset (harmony)
            externalPitchMod += amount * 12.0f;
            break;

        case CouplingType::AudioToFM:
        {
            // Source audio becomes FM modulator for ORBITAL's partials
            // Per-sample: stored in buffer, consumed in render loop
            externalFmMod = amount;
            if (sourceBuffer != nullptr)
            {
                float* cL = couplingAudioBuf.getWritePointer(1);
                for (int n = 0; n < numSamples; ++n)
                    cL[n] = sourceBuffer[n] * amount;
            }
            break;
        }

        case CouplingType::AudioToRing:
        {
            // Ring modulation: ORBITAL output × source audio
            if (sourceBuffer != nullptr)
            {
                float* rL = couplingRingBuf.getWritePointer(0);
                for (int n = 0; n < numSamples; ++n)
                    rL[n] = sourceBuffer[n] * amount;
            }
            break;
        }

        case CouplingType::EnvToDecay:
            // Source envelope → group decay time multiplier
            externalDecayMod = amount;
            break;

        case CouplingType::RhythmToBlend:
            // Source rhythm → harmonic group enable animation
            externalBlendMod = amount;
            break;

        case CouplingType::AmpToChoke:
            // NOT accepted — kills all harmonics, no musical use
            break;

        case CouplingType::FilterToFilter:
            // NOT accepted — formant filter is spectral domain, not time domain
            break;

        default:
            break;
    }
}
```

### 8.3 AudioToWavetable — The Spectral Coupling Algorithm

This is the core coupling feature. External audio's harmonic content reshapes ORBITAL's partial amplitudes.

**Algorithm:** Once per block, analyze the coupling audio buffer to extract harmonic energy at each partial's frequency. Use this to create amplitude offsets for each partial.

```
For each block:
    1. Compute RMS energy of coupling audio → couplingRMS
    2. If couplingRMS > threshold:
       For each partial k:
           // Measure energy near partial k's frequency using a
           // simple IIR bandpass (1-pole resonator tracking partial freq)
           // This is NOT an FFT — too expensive for real-time.
           // Instead: accumulate abs(coupling * sin(partial_freq * t))
           // over the block → energy estimate at that harmonic
           spectralEnergy[k] += abs(couplingAudio[n] * fastSin(partialPhaseRef[k]))
       3. Normalize spectralEnergy to [0..1]
       4. Apply as additive offset to profile amplitudes:
           effectiveAmp[k] = baseAmp[k] + couplingAmount * spectralEnergy[k]
```

**Cost:** 64 × 1 fastSin × blockSize correlations per block. At 512 samples: 64 × 512 = 32,768 multiplies per block = ~64 multiplies per sample. Acceptable.

**Alternative (simpler, for v1):** Just use the source audio's amplitude envelope to modulate all partials uniformly, weighted by the source's spectral content approximated through zero-crossing rate. This is cheaper and still creates the "DNA transfer" effect:

```cpp
// Simple spectral coupling (v1):
// High zero-crossing rate → boost upper partials
// Low zero-crossing rate → boost lower partials
float zcr = computeZeroCrossingRate(couplingAudioBuf, numSamples);
float brightness = clamp(zcr / 10000.0f, 0.0f, 1.0f);  // normalize
for (int k = 0; k < 64; ++k)
{
    float weight = (brightness > 0.5f)
        ? static_cast<float>(k) / 63.0f      // bright source → upper partials
        : 1.0f - static_cast<float>(k) / 63.0f;  // dark source → lower partials
    spectralCouplingOffset[k] = couplingRMS * weight * amount;
}
```

### 8.4 getSampleForCoupling()

```cpp
float getSampleForCoupling(int channel, int sampleIndex) const override
{
    auto si = static_cast<size_t>(sampleIndex);
    if (channel == 0 && si < outputCacheL.size()) return outputCacheL[si];
    if (channel == 1 && si < outputCacheR.size()) return outputCacheR[si];
    // Channel 2: envelope level for AmpToFilter/AmpToChoke routes
    if (channel == 2) return envelopeOutput;
    return 0.0f;
}
```

Uses `outputCacheL/R` vector pattern (like ODDFELIX), written per-sample in the render loop. Enables tight per-sample coupling for AudioToFM and AudioToRing routes.

## 10. Macro Implementation

### 10.1 applyMacros()

Called once per block after parameter snapshot, before render loop.

```cpp
void applyMacros(float& brightness, float& oddEven, float& morph,
                 float& stereoSpread, float& formantShift,
                 float grpAtk[4], float grpDec[4]) noexcept
{
    float spectrum  = p_macroSpectrum->load();
    float evolve    = p_macroEvolve->load();
    float coupling  = p_macroCoupling->load();
    float space     = p_macroSpace->load();

    // M1 SPECTRUM: 0 = dark fundamental only, 1 = all harmonics blazing
    if (spectrum > 0.001f || spectrum < 0.999f)
    {
        brightness = spectrum;
        // At low spectrum: emphasize odd harmonics (warmer)
        // At high spectrum: equal odd/even (brighter)
        oddEven = 0.3f + spectrum * 0.4f;
    }

    // M2 EVOLVE: 0 = Profile A, static, fast envelopes
    //            1 = Profile B, evolving, slow envelopes
    if (evolve > 0.001f)
    {
        morph = std::max(morph, evolve);
        // Slow down group envelopes as EVOLVE increases
        float envScale = 1.0f + evolve * 4.0f;  // up to 5× slower
        for (int g = 0; g < 4; ++g)
        {
            grpAtk[g] *= envScale;
            grpDec[g] *= envScale;
        }
    }

    // M3 COUPLING: controls external engine influence depth
    if (coupling > 0.001f)
    {
        // Shift formant up as coupling increases → spectral contrast
        formantShift = std::max(formantShift, coupling * 12.0f);
    }

    // M4 SPACE: stereo spread + reverb send
    if (space > 0.001f)
    {
        stereoSpread = std::max(stereoSpread, space);
    }
}
```

### 11.1 Post Filter

```cpp
CytomicSVF postFilterL;
```

```cpp
void prepare(double sampleRate, int maxBlockSize) override
{
    sr = sampleRate;
    srf = static_cast<float>(sampleRate);
```

## 13. Voice Stealing

Oldest-note stealing with 5ms crossfade (matching ODDFELIX's pattern):

```cpp
int findFreeVoice()
{
    // Find inactive voice
    for (int i = 0; i < kMaxVoices; ++i)
        if (!voices[i].active)
            return i;

    // LRU: steal oldest
    int oldest = 0;
    uint64_t oldestTime = UINT64_MAX;
    for (int i = 0; i < kMaxVoices; ++i)
    {
        if (voices[i].startTime < oldestTime)
        {
            oldestTime = voices[i].startTime;
            oldest = i;
        }
    }
    return oldest;
}
```

Crossfade on steal:
```cpp
if (voices[idx].active)
    voices[idx].fadeOutLevel = voices[idx].envLevel;
else
    voices[idx].fadeOutLevel = 0.0f;
```

Applied per-sample: `stealFade = 1.0f - fadeOutLevel` where `fadeOutLevel` decreases by `1.0f / (0.005f * srf)` per sample.
