## Process Gaps Found Tonight

- Spot-check Entangled presets: any with `couplingMode: null` or fewer than 2 engines is suspect.

## Skill / Process Improvements

- **`/audit-prefix`**: A lightweight skill that scans `prefixForEngine()` cases against the registered engine list and flags gaps. Should run automatically at the top of any session that installs a new engine.
- **`/memory-sync`**: Skill that re-counts engines, presets, and mood distribution from source and patches MEMORY.md counts in one pass. Run at end of every install session.
- **Preset authoring checklist**: Before any preset batch lands, confirm: (1) names unique within repo, (2) mood category structurally valid, (3) coupling fields populated for Entangled entries.
