4. File a bug for any item that cannot be checked off.

## Section 5 — MIDI Verification (Issue #101)

These tests verify MIDI contract correctness fleet-wide.  Run with any engine that has standard MIDI implementation.

### Pitch Bend
- [ ] Pitch bend range — play note, bend wheel full up: should be exactly +2 semitones
- [ ] Pitch bend range — play note, bend wheel full down: should be exactly -2 semitones
- [ ] Pitch bend return — release wheel: returns precisely to center (no residual detune)

### Sustain Pedal (CC#64)
- [ ] Basic sustain — hold pedal, play note, release note, release pedal: note sustains then releases cleanly
- [ ] Multi-note sustain — hold pedal, play 8 notes rapidly, release pedal: all 8 notes release, no stuck voices
- [ ] Sustain interlock — hold pedal, play note, press pedal again while held: no double-trigger or click

### All Notes / All Sound Off
- [ ] All Notes Off (CC#123) — play chord, send CC#123: immediate silence, all voices stop
- [ ] All Sound Off (CC#120) — play chord with reverb tail active, send CC#120: immediate silence including FX tails

### Expression & Modulation
- [ ] Channel pressure / aftertouch — if supported: play note, apply aftertouch: smooth, continuous response
- [ ] Mod wheel (CC#1) — sweep from 0 to 127: maps to expected parameter, smooth response, no zipper noise

### Voice Integrity
- [ ] Voice stealing — exceed polyphony limit rapidly: oldest/quietest voice stolen gracefully, no crash
- [ ] Note off timing — play rapid staccato notes at 240 BPM: no orphaned notes

- [ ] 4-engine coupling matrix — load 4 engines in coupling matrix: CPU stays below 60%, no xrun/dropout
- [ ] Rapid preset switching — click through 20 presets in succession quickly: no crash, no frozen UI
