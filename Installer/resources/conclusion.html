Installation complete!

XOceanus is ready to use.

Next steps

  1. AU Plugin users — open your DAW and run a plugin scan (or re-scan the AU plug-ins folder). XOceanus will appear under XO_OX Designs in your instrument list.
  2. Standalone users — launch XOceanus from your Applications folder or Launchpad.
  3. Load one of the 16 mood preset categories and start exploring. The COUPLING macro routes modulation between engines in real time.

AU identification code

If your DAW requires manual entry, use the following Audio Unit code to locate XOceanus:

aumu Xolk XoOx

To validate the AU from Terminal run:

auval -v aumu Xolk XoOx
